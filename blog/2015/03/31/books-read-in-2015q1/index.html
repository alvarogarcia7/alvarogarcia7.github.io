Books Read in 2015Q1

Mar 31st, 2015 10:49 am

Books I’ve read this quarter Q1 on 2015, as inspired by Manuel, on this post:

  • Fifty Quick Ideas to Improve your User Stories, by Gojko Adzic and David Evans
  • Fun retrospectives by Taina Caetano and Paulo Caroli
  • Implementation patterns by Kent Beck
  • Clean code by Robert C. Martin; again

Books started, not yet finished (WIP):

  • The Well-Grounded Java Developer, by Benjamin J. Evans and Martijn Verburg
  • Refactoring, by Martin Fowler, (with Kent Beck, John Brant, William Opdyke, and Don Roberts)
  • Growing Object-Oriented Software Guided by Tests, by Steve Freeman and Nat Pryce
  • Flow, by Mihaly Csikszentmihalyi

(Note: I’ve written this list a posteriori, in May 2015)
